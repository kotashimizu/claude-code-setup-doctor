# リリースノート — v0.1.0-beta

**リリース日**: 未定（テスト完了後）

---

## このリリースについて

Claude Code Setup Doctor v0.1.0-beta は Windows 向けの非公式コンパニオンツールの最初のベータリリースです。

**注意**: このツールは Anthropic 社の製品ではありません。非公式ツールです。

---

## 動作環境

| 要件 | バージョン |
|------|-----------|
| OS | Windows 10 (Build 17763 以降) または Windows 11 |
| アーキテクチャ | x64 または ARM64 |
| メモリ | 4 GiB 以上 |
| .NET ランタイム | 不要（自己完結型単一ファイル） |

---

## 診断チェック一覧

### 必須
- CHK-OS-001: OS バージョン（Windows 10 Build 17763 以降）
- CHK-ARCH-001: CPU アーキテクチャ（x64 / ARM64）
- CHK-MEM-001: 物理メモリ（4 GiB 以上）
- CHK-SHELL-001: Windows PowerShell
- CHK-CLAUDE-001: Claude CLI の検出
- CHK-CLAUDE-002: Claude CLI のバージョン確認
- CHK-PATH-001: User PATH への Claude パスの追加

### 推奨
- CHK-SHELL-002: PowerShell 7
- CHK-GIT-001: Git for Windows
- CHK-GITBASH-001: Git Bash
- CHK-AUTH-001: Claude 認証状態
- CHK-CLAUDE-003: Claude Desktop との競合検出
- CHK-CLAUDE-004: 複数インストールの検出

### 情報のみ
- CHK-NET-001/002/003: Anthropic API エンドポイントへの到達性
- CHK-WINGET-001: winget の存在
- CHK-DOCTOR-001: claude doctor 補助確認

---

## 自動修復対応

| ID | 内容 |
|----|------|
| REM-PATH-CLAUDE-LOCALBIN | `%USERPROFILE%\.local\bin` を User PATH に追加 |
| REM-CLAUDE-GITBASH-SETTING | `CLAUDE_CODE_GIT_BASH_PATH` を `~/.claude/settings.json` に設定 |

---

## セキュリティに関する注意

- 修復前にバックアップを作成します
- Machine PATH / HKLM レジストリは変更しません
- APIキー・トークン・パスワードは収集・表示しません
- レポートはリダクション処理済みです

- REM-INSTALL-CLAUDE-WINGET（winget 経由インストール）は v0.2.0 予定
- ARM64 環境でのエミュレーション動作は未検証
