# 00. Project Overview

## 0.1 背景

WindowsでClaude Codeを初めて導入する利用者は、インストール自体が完了していても、PATH、シェル、複数インストール、デスクトップアプリとのコマンド競合、認証、企業ネットワーク制約などで起動できない場合があります。

現場サポートでは、利用者ごとに環境差を確認し、コマンドを実行し、環境変数や設定ファイルを修正する必要があります。本プロダクトは、この診断を標準化し、利用者自身または情シス担当者が原因を把握できる状態にします。

## 0.2 前提の修正

初期アイデアでは「Gitを正しい場所へ配置し、PATHを修正するラッパー」を想定していました。しかし現行仕様では次の通りです。

- Git for Windowsは任意です。Git Bashを提供しますが、未導入時はPowerShellを利用できます。
- ネイティブ版Claude CodeはNode.jsを必須としません。
- Claude Code本体のPATH不備は `%USERPROFILE%\.local\bin` の不足として扱います。
- Gitが存在するがClaude Codeから見えない場合は、`CLAUDE_CODE_GIT_BASH_PATH` の設定が公式に案内されています。

したがって、プロダクトの中心は「Git配置ツール」ではなく「Claude Codeセットアップ診断・限定修復ツール」です。

## 0.3 プロダクト定義

**仮称:** Claude Code Setup Doctor  
**カテゴリ:** Windowsデスクトップ診断ユーティリティ  
**位置づけ:** 非公式コンパニオンアプリ

### 解決する問題

- Claude Codeがインストール済みか判断できない。
- `claude` コマンドが見つからない。
- PATHに複数の `claude` があり、別の実行ファイルが優先される。
- Claude Desktopの古いApp Execution AliasがCLIより優先される。
- PowerShellとGit Bashのどちらも見つからない。
- GitはあるがGit Bashの場所をClaude Codeが認識しない。
- 認証が完了していない。
- 企業ネットワークやEDRが原因だが、利用者にはPATH問題に見える。

### 解決しない問題

- Claudeアカウント契約、座席割当、請求、組織権限の付与
- 企業ネットワークポリシーの自動変更
- 開発プロジェクトの作成やGit初期化
- Claude Code本体のチャット／コード編集UI
- コードベースの安全性や品質診断

## 0.4 対象利用者

### Persona A: 非エンジニア利用者

- PowerShellやPATHの意味を理解していない。
- 指示されたアプリをダブルクリックして結果を確認したい。
- エラー時には「何を誰に伝えるか」が必要。

### Persona B: 導入支援者

- 複数顧客に同一手順を提供したい。
- リモート会議なしで一次診断を完了させたい。
- 診断レポートを受け取り、二次対応を判断したい。

### Persona C: 情報システム担当者

- PATH、GPO、EDR、プロキシ、証明書の切り分けが必要。
- 利用者アプリが勝手に管理領域を変更しないことを重視する。
- 端末情報を過剰収集しないレポートが必要。

## 0.5 対象環境

- Native WindowsをMVP対象とする。
- Claude Code公式要件に合わせ、Windows 10 1809以降、Windows Server 2019以降、64-bit、4 GB以上を判定する。
- x64を初回配布対象とする。ARM64は設計上考慮し、後続リリースで配布する。
- WSLは検出して案内できるが、自動修復対象外とする。
- Claude Desktop Codeタブは同じ基盤を使用するが、MVPの診断基準はCLIを中心とする。

## 0.6 プロダクト原則

1. **診断優先:** まず事実を検出し、推測を結果に混ぜない。
2. **最小変更:** 変更範囲をUser PATHとユーザー設定に限定する。
3. **承認制:** 変更内容を表示し、承認された修復だけ実行する。
4. **再診断:** コマンド成功ではなく、修復後チェックの成功を完了条件とする。
5. **企業環境尊重:** 管理ポリシーを回避しない。IT対応に引き継ぐ。
6. **機密情報最小化:** トークン、APIキー、完全PATH、ユーザー名を収集しない。
7. **公式仕様追従:** 既定パス、コマンド、要件を設定データとして管理し、更新可能にする。
